Create Hyron app with no build configuration

# Installation

```sh
$ npm install -g hyron-cli
```

# Usage

## 1. Init a module/app used template
```
hyron init <app|services|plugins|addons>
```

## 2. Start app using JSON build file
```
hyron start [path]
```
